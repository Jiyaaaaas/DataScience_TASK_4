# Traffic Accident Data Analysis

This project analyzes traffic accident data to identify patterns related to:

- Road Conditions
- Weather Conditions
- Time of Day
- Accident Hotspots

- The dataset used in this project is too large to upload to GitHub.

Original Dataset Source:

US Accidents Dataset (Kaggle)

URL: https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents

Python Libraries that are required: 
- Pandas
- Numpy
- Matplolib
- Seaborn
- Pyplot
- folium
